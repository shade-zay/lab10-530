Lab 10 – Framework Page

Materialize CSS Framework Page

This page was created for Lab 10 using the Materialize CSS framework. It allowed me to quickly build a simple layout without writing complex CSS from scratch.

How I Installed Materialize

I used the CDN version of Materialize:

  • Opened the Materialize website and went to the “Getting Started” page.
  • Copied the CSS CDN link into the head of this page.
  • Added the JavaScript CDN link at the bottom of the page.
  • Saved the page and tested it in the browser.
How I Built This Page

I used basic Materialize components:

  • A navigation bar with the nav and nav-wrapper classes.
  • A two-column layout using the grid system.
  • Cards to organize the explanations.
  • Chips to summarize the tools used.
Difficulties and How I Solved Them

I had to get used to the grid system because the class names were different from what I normally use. After a few tries, I understood how the columns resize depending on the screen size.

I also tested different colour combinations to keep the text readable. Using the browser inspector helped adjust the styling until it looked balanced.

Another small issue was forgetting to include the JavaScript CDN at first, which caused some components not to load, but adding the script fixed it instantly.
